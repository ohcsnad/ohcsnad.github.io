### Web Exercise 1: DATA SET

[RedFin Housing Market Data](https://www.kaggle.com/datasets/andradaolteanu/redfin-housing-market-prices)

This data set from RedFin provides insight to the US Housing Market. The RedFin data center provides data on the housing and rental market in the US by specific regions from the national level all the way down to the neighborhood level. There is a lot of potential here for analysis because this data is consistently updated with the latest changes to rentals and homes on market by MLS. For example, we can perform analysis on the general housing market at the national level to see the median price of homes by the type of homes. In addition, if we were interested in finding the best communities to rent or purchase or home, we would do analysis of homes in different zip codes in specific cities.

### Web Exercise 1: Data Science Book

[Modeling with Data](https://modelingwithdata.org/about_the_book.html)

The data science book I have chosen is "Modeling with Data" by Ben Klemens. Ben Klemens decided to create this book because he believed that there needed to be a balance between statistical theory and practical application of statistical computing. I also believe that the best way to properly digest statistical concepts is to apply them in various projects. This book reviews different statistical methods, tools for data management and analysis, and then demonstrates how to apply these two together.
